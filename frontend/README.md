# Университетский Telegram-бот

Веб-интерфейс для Telegram-бота для студентов российских вузов.

## Описание

Минималистичный и современный веб-интерфейс для управления различными функциями университета через Telegram-бота. Поддерживает несколько ролей пользователей с разными возможностями.

## Структура проекта

- `index.html` - Главная страница с выбором вуза
- `role-selection.html` - Страница выбора роли пользователя
- `dashboard-student.html` - Дашборд для студентов
- `dashboard-abiturient.html` - Дашборд для абитуриентов
- `dashboard-staff.html` - Дашборд для сотрудников вуза
- `dashboard-admin.html` - Дашборд для руководителей вуза
- `free-classrooms.html` - Специальная страница свободных аудиторий (для РЭУ им. Плеханова)
- `styles.css` - Общие стили для всего приложения

## Функционал по ролям

### Абитуриенты
- Информация о вузе
- Образовательные программы
- Условия поступления
- Запись на день открытых дверей
- Запись на экскурсию

### Студенты
- **Обучение**: Расписание, обратная связь, факультативы
- **Проектная деятельность**: Создание проектов, поиск команды, доступные проекты
- **Карьера**: Консультации, вакансии
- **Деканат**: Справки, оплата обучения, документы
- **Общежитие**: Оплата, услуги, пропуска, техподдержка
- **Внеучебная деятельность**: Календарь событий
- **Библиотека**: Заказ книг, электронная библиотека
- **Свободные аудитории** (только для РЭУ им. Плеханова)

### Сотрудники вуза
- Командировки
- Отпуска
- Офисные услуги
- Внеучебная деятельность

### Руководители вуза
- Управление кампусом
- Академические показатели
- Новостной агрегатор

## Дизайн

- Минималистичный и современный интерфейс
- Адаптивный дизайн для мобильных устройств
- Использование современных CSS-переменных
- Плавные анимации и переходы
- Интуитивная навигация

## Технологии

- HTML5
- CSS3 (с CSS-переменными)
- Vanilla JavaScript
- LocalStorage для хранения состояния

## Запуск

Просто откройте `index.html` в браузере. Для полной функциональности рекомендуется использовать локальный веб-сервер.
